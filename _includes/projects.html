People Data application
2013-2014
- Online application to provide information on students and alumnus.

  • Worked single-handedly on internal web app, that provides live data from numerous job sites and academic sites about the students and alumnus.
  • Implemented data extraction modules in form of Django models and external controller classes.
  • Implemented multiple collections in MongoDB for hierarchical storage and fast access modules using PyMongo.
  • Normalization of the models in Django, by defining separate Documents such as for education, pictures, experiences, personal info etc.
  • Implemented separate controllers for authorizations, connection establishment, database operations and API integrations.
  • Strictly focused on MVC principles and loose coupling of modules.
